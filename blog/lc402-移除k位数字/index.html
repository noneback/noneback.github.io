Today’s LeetCode daily challenge is LC402: Remove K Digits. I completed it after reviewing the solution, so I’m noting down the thought process.

Description

The problem is described as follows:

Given a non-negative integer num represented as a string, remove k digits from the number so that the remaining number is the smallest possible.

Note:

  • The length of num is less than 10002 and is greater than or equal to k.
  • num does not contain any leading zeros.
Example 1:

Input: num = "1432219", k = 3
Output: "1219"
Explanation: Remove digits 4, 3, and 2 to form the smallest number "1219".

Analysis

The key to this problem is determining which digits need to be discarded to form the smallest possible number.

To achieve this, we need to:

  1. Compare the sizes of the digits.
  2. Determine whether a digit should be discarded.

Comparing the Sizes of Digits

When comparing two numbers, we determine their sizes based on the highest differing digit.

  1. First, compare the number of digits. If they are different, the number with more digits is larger.

a = 12xxx, b = 11xxxx, clearly a < b

  1. If the number of digits is the same, compare digit by digit from left to right until a difference is found.

a = 12xxxxx, b = 13xxxxx, since 2 < 3 at the second digit, we have a < b.

Determining Whether a Digit Should Be Discarded

Conditions for Discarding

The greedy strategy involves the following:

For a number $D_1D_2…D_x$, if there is a position $i$ such that $D_i < D_{i-1}$, discard $D_{i-1}$.

If no such position exists, the entire sequence is non-decreasing (e.g., 123456789). In this case, simply remove the last digit.

Let’s simulate the given example:

Suppose num = 1432219, k = 3. Iterate through num, and each time compare the current digit with the one to its left. If it is smaller, discard the left digit; otherwise, push it to the stack.

The result must be a non-decreasing sequence, so we use a monotone stack to store the digits that are not discarded.

If fewer than k digits have been discarded by the end, discard the remaining digits from the end.

Code

/*
 * @lc app=leetcode.cn id=402 lang=cpp
 *
 * [402] Remove K Digits
 */

// @lc code=start
// Monotone Stack + Greedy
#include <bits/stdc++.h>
using namespace std;
class Solution {
 public:
  string removeKdigits(string num, int k) {
    vector<char> ret;

    for (auto& ch : num) {
      while (ret.size() > 0 && k && ret.back() > ch) {
        ret.pop_back();
        k--;
      }
      ret.push_back(ch);
    }
    // Handle remaining k, by removing the last digits since the sequence is non-decreasing

    while(k--){
        ret.pop_back();
    }

    string ans = "";
    bool begWithZero = true;
    for (auto& digit : ret) {
      if (digit == '0' && begWithZero) {
        continue;
      }
      begWithZero = false;
      ans += digit;
    }

    return ans == "" ? "0" : ans;
  }
};
// @lc code=end
